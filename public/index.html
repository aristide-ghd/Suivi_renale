Bienvenue sur l'API Suivi Rénal

À propos de l'API

L'API Suivi Rénal est une solution complète pour la gestion du suivi des patients atteints de maladies rénales. Elle offre des outils puissants pour les professionnels de santé afin de suivre les données médicales, améliorer la prise en charge des patients et optimiser les soins.

Cette API est conçue pour être sécurisée, performante et facile à intégrer dans vos systèmes existants.

Guide pour utiliser la documentation Swagger :

  • Accédez à la documentation en cliquant sur le bouton ci-dessus ou en visitant /api-docs.
  • Explorez les différentes routes disponibles pour les utilisateurs, administrateurs, médecins, infirmiers et patients.
  • Pour tester une route, cliquez sur celle-ci, entrez les paramètres requis, et cliquez sur "Try it out".
  • Utilisez un token JWT valide pour accéder aux routes protégées. Vous pouvez le fournir dans la section "Authorize".
  • Consultez les réponses possibles pour chaque route, y compris les codes de statut HTTP et les exemples de réponses.
