Bienvenue sur l'API Suivi Rénal

L'API Suivi Rénal est une solution complète pour la gestion du suivi des patients atteints de maladies rénales. Elle offre des outils puissants pour les professionnels de santé afin de suivre les données médicales, améliorer la prise en charge des patients et optimiser les soins.

Cette API est conçue pour être sécurisée, performante et facile à intégrer dans vos systèmes existants.

Accéder à la documentation

Guide pour utiliser la documentation Swagger :

  • 1. Accéder à la documentation : Rendez-vous sur /api-docs dans votre navigateur.
  • 2. Explorer les routes : Les routes sont regroupées par catégories (ex. Users, Doctors, Patients). Cliquez sur une catégorie pour afficher les routes associées.
  • 3. Tester une route : Cliquez sur une route pour afficher ses détails, puis cliquez sur "Try it out". Remplissez les paramètres requis et cliquez sur "Execute" pour voir la réponse.
  • 4. Utiliser un token JWT : Cliquez sur le bouton "Authorize" en haut à droite, entrez votre token JWT, puis cliquez sur "Authorize".
  • 5. Consulter les réponses : Après avoir exécuté une requête, consultez la section "Response" pour voir les résultats et les codes HTTP.
